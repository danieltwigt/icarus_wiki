Preparing ICARUS
================

.. _Valve Index Setup Guide: https://steamcdn-a.akamaihd.net/steam/apps/1059530/manuals/KitQSG_EN.pdf
.. autosummary::
   :toctree: generated

To start ICARUS, follow these steps:

Step 1: system requirements
---------------------------

If not done yet, ensure that the system requirements as described in Hey, read the :ref:`system_requirements` section are met.

Step 2: dependancies
--------------------

If not done yet, ensure that dependancies as described in :ref:`system_requirements` are installed.

Step 3: Valve Index setup
-------------------------

If not done yet, ensure that the Valve Index is setup following instructions as provided in the `Valve Index Setup Guide`_, also accessible below.

.. raw:: html

  <embed src="https://steamcdn-a.akamaihd.net/steam/apps/1059530/manuals/KitQSG_EN.pdf" width="800" height="400" type="application/pdf">

Step 4: Unpacking ICARUS
------------------------

If not done yet, unpack ICARUS as included in the zip-file ICARUS_<build_number>.zip to a location of choice on the PC.

Step 5: Starting ICARUS
-----------------------

Start ICARUS by running the executable <installation location>\WindowsNoEditor\HADES.exe.
